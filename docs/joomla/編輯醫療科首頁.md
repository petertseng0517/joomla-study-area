# 步驟說明
1. 進入SP Page管理介面：元件-> SP Page Builder->Pages
2. 分類選「醫療科」：Category-> 醫療科
3. 開啟前端編輯器：選定要編輯的醫療科首頁，開啟該頁面的前端編輯器(Frontend Editor)
4. 更新頁面內容：點選想編輯的區塊後，在功能區塊進行編輯
5. 存檔：點選「存檔並關閉」
