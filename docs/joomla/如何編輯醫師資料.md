# 步驟說明
1. 內容->文章->篩選（醫療科、標籤：醫師）
2. 開啟文章檔案->設定好醫師姓名、醫師編號、分類(醫療科名稱)、標籤(00-醫師)，點選SP編輯器->存檔。
3. 開啟sp 前端編輯器，進行醫師資料編輯。
4. 提醒：醫師照片檔案，請聯繫公傳安俞協助後製上傳。(檔案命名：dr-醫師編號)，存檔位置：/images/00MD/dr/
5. 切換到「圖片和連結」分頁，也設定醫師照片縮圖。
