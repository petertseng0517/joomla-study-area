# 步驟說明
1. 先將圖片在本機端準備好，檔案大小建議不超過2MB
2. 開啟Joomla的文章編輯器，點插入圖片的icon，開啟功能對話方塊，將圖片上傳到伺服器。
3. 將圖片插入到指定位置
4. 調整圖片大小。
5. 提醒：切勿直接從Notes上複製圖片直接貼到Joomla編輯器上，會導致系統效能變慢。
