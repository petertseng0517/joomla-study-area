- 作者：曾建瑋(花蓮慈濟醫院公共傳播室多媒體組網管, 分機 #15295)

---
# 禁忌
1. 切勿直接將Notes系統的內容，直接複製貼到Joomla的文章中。
2. 切勿直接將Word檔案的內容，直接複製貼到Joomla的文章中。
3. 切勿看到系統升級、套件更新的按鈕，就自行按下去。
4. 切勿將個人帳號借給他人使用。
5. 編輯文章後，請記得正常存檔離開(SAVE and CLOSE)。

---
# 說明
1. 如果直接複製Notes系統內容，貼到Joomla文章中，會導致：
    - 帶入太多Notes專用的格式碼，他不屬於網頁正式的程式碼，會導致網站效能降低。
    - 有可能複製到院內的連結，該連結在院外瀏覽網頁時不能使用，但瀏覽器會因為不斷測試該連結當掉。
    - 從Notes直接複製貼上的圖片，並非正常的圖片檔。他是一大群Notes系統路徑的程式碼，也會導致網站效能降低。
```
正確做法：請將圖片下載到本地電腦後，再上傳到Joomla管理後台，文章插入該圖片使用。
正確做法：複製的文字內容，先貼到記事本(變成純文字)，再貼到joomla的文章內文中。
```
