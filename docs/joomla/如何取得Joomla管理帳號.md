# 步驟說明
1. 點選：醫院Notes系統 》 各類申請單 》 公共傳播類(網站維護及製作申請) 》建立新申請
2. 申請項目：維護頁面內容
3. 網址欄位：請填入要申請管理帳號的網站網址
4. 說明欄位：申請管理權限帳號。(請提供資訊，包括：姓名, 分機, email, 帳號, 申請緣由)
5. 審核通過後，會將帳密寄到電子信箱
