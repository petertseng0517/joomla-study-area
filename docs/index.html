每一個故事
都值得被看見
從急診室的燈光到病房裡的守候， 從醫護人員的堅持到病人的重生， 我們走進生命現場， 用文字與影像留下這份溫度。
慈濟人文
醫療之外
是人與人的相遇
慈濟人文不只是精神， 它活在每一次問診的眼神裡、 每一個跨越病痛的擁抱裡。 我們的任務，是讓這份精神被更多人看見。
研究與成就
醫學的進步
從花蓮出發
花蓮慈濟醫院的醫療團隊持續突破， 研究成果走向國際。 我們負責將這些成就轉化為社會能理解的語言， 讓醫學的光照得更遠。
