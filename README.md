# 我的個人Joomla教學筆記庫

我使用 Obsidian 撰寫，並發佈到 GitHub Pages 的個人知識庫。

## 如何瀏覽我的筆記

這個知識庫是透過 Obsidian 與 GitHub Pages 發佈，您可以：

1.  **直接在 GitHub 瀏覽**：點擊檔案列表，瀏覽各個 Markdown 文件。
2.  **透過 GitHub Pages 網站瀏覽**：點擊專案首頁的 **`GitHub Pages 連結`**，體驗更佳的閱讀介面。
3.  **在本地端使用 Obsidian**：如果您是 Obsidian 用戶，可以將這個專案 `git clone` 到您的電腦，並以 Obsidian 軟體開啟，體驗完整的雙向連結功能與圖形化視圖。
