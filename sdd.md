**狀態：** 草稿

## 1. 目的與背景

慈濟醫院同仁上傳圖片至 Joomla 前，需自行處理圖片格式與大小（限制 2MB）。目前教學文件已有警告，但缺乏輔助工具，同仁仍需另找第三方軟體。

**目標：** 在現有 Joomla 教學網站（MkDocs / GitHub Pages）內，提供一個零安裝的圖片裁切 + 轉 .webp 工具頁面，降低同仁的操作門檻。

---

## 2. 需求規格

### 2.1 功能需求

| ID | 需求描述 | 優先級 |
|----|----------|--------|
| F-01 | 使用者可拖曳或點選上傳圖片（支援 jpg / png / gif / webp） | 必要 |
| F-02 | 圖片上傳後顯示裁切介面 | 必要 |
| F-03 | 提供固定比例按鈕：`1:1`、`4:3`、`16:9`、`3:4` | 必要 |
| F-04 | 提供「自由裁切」模式（無比例限制） | 必要 |
| F-05 | 提供品質滑桿（0–100%，預設 85%） | 必要 |
| F-06 | 顯示預估輸出檔案大小（KB） | 建議 |
| F-07 | 點擊「下載」後產生 .webp 檔案並儲存到本機 | 必要 |
| F-08 | 頁面提示此工具的用途與下一步（上傳到 Joomla） | 建議 |

### 2.2 非功能需求

| ID | 需求描述 |
|----|----------|
| NF-01 | 純前端實作，不需後端伺服器，所有處理在瀏覽器本機完成 |
| NF-02 | 支援 Chrome / Edge（醫院 Windows 環境主流瀏覽器） |
| NF-03 | 不引入 npm / build 工具，所有依賴透過 CDN 載入 |
| NF-04 | 單一 HTML 檔案，方便維護與部署 |
| NF-05 | Safari 17+ 可用；舊版 Safari 顯示不支援提示（fallback） |

## 4. 介面規格

### 4.1 上傳區（Drop Zone）

- `id="drop-zone"`
- 接受事件：`dragover`、`drop`、`click`（觸發 `input[type=file]`）
- 接受的 MIME 類型：`image/jpeg, image/png, image/gif, image/webp`
- 上傳後隱藏 drop-zone，顯示裁切區

### 4.2 Cropper.js 初始化參數

```js
new Cropper(imageElement, {
  viewMode: 1,          // 限制裁切框在圖片範圍內
  autoCropArea: 0.8,    // 預設裁切 80% 範圍
  movable: true,
  zoomable: false,      // 關閉縮放，避免操作混亂
  aspectRatio: NaN,     // 預設自由裁切，按鈕切換
});
```

### 4.3 比例切換按鈕

| 按鈕文字 | `aspectRatio` 值 |
|----------|-----------------|
| 1:1 | `1` |
| 4:3 | `4/3` |
| 16:9 | `16/9` |
| 3:4 | `3/4` |
| 自由 | `NaN` |

### 4.4 轉換與下載邏輯

```
使用者點擊「下載」
  → cropper.getCroppedCanvas() 取得裁切後 canvas
  → canvas.toBlob(callback, 'image/webp', quality)
    → quality = 品質滑桿值 / 100
  → 建立 <a> 標籤，href = objectURL，download = 原檔名.webp
  → 觸發 click，自動下載
  → 更新預估大小顯示
```

### 4.5 預估大小顯示

- 每次品質滑桿變更時，呼叫 `canvas.toBlob()` 取得 Blob size
- 顯示格式：`約 XX KB`

## 6. 測試計畫

| 測試項目 | 步驟 | 預期結果 |
|----------|------|----------|
| 本機開發預覽 | `mkdocs serve`，開啟工具頁面 | 頁面正常顯示 |
| 上傳 jpg | 拖曳 jpg 到上傳區 | 顯示 Cropper 裁切介面 |
| 比例切換 | 點擊 `16:9` 按鈕 | 裁切框鎖定 16:9 |
| 自由裁切 | 點擊「自由」按鈕 | 裁切框可任意調整 |
| 品質調整 | 滑動品質滑桿至 50% | 預估大小縮小，下載檔案品質降低 |
| 下載 | 點擊「下載 .webp」 | 瀏覽器儲存 .webp 檔案，副檔名正確 |
| 檔案大小 | 上傳 2MB jpg，下載 | .webp 應明顯小於原始 jpg |
| 舊版 Safari | 在不支援 WebP output 的環境開啟 | 顯示「您的瀏覽器不支援」提示 |
